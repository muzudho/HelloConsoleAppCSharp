# ソリューション一式の作成


## フォルダーとファイルの構成

例えば、わたしは以下のようなフォルダー構成でプロジェクトを管理しています。  

```plaintext
📁 E:                    # ドライブのルート。 C ドライブでもいいし、D ドライブでもいいし、どこでもいいです。
+-- 📁 github.com        # フォルダーを新規作成する。GitHub のユーザーネームのフォルダーを作ると、あとで管理しやすいと思います。
    +-- 📁 muzudho       # わたしの名前。
```


## ［HelloConsoleAppCSharp］プロジェクトの作成

Visual Studio 2026 を起動し、[新しいプロジェクトの作成］をクリックしてください。

プロジェクト テンプレートから［コンソール アプリ］を選択してください。  
（フィルターを `C#`, `Windows`, `コンソール` にすると見つけやすいと思います）  
このとき、 `.NET Framework` ではなく、 `.NET` の方を選択してください。  

プロジェクトの名前は `HelloConsoleAppCSharp` としてください。  

`ソリューションとプロジェクトを同じディレクトリに配置する` オプションは、チェックを入れないでください。  

ソリューションの名前は、（プロジェクト名と同じ） `HelloConsoleAppCSharp` としてください。  
（この名前は、Git Hub のリポジトリーの名前と同じにする想定です）  

同じ名前を２度入力するのは気になりますが、あとで楽なのでこうしてください。  

フレームワークは、最新の長期的なサポートのもの（例えば .NET 10.0）を選択してください。  

👇　以下のような構成でプロジェクトが作成されると思います。  

```plaintext
📁 E:									    # ドライブのルート。 C ドライブでもいいし、D ドライブでもいいし、どこでもいいです。
+-- 📁 github.com						    # フォルダーを新規作成する。GitHub のユーザーネームのフォルダーを作ると、あとで管理しやすいと思います。
    +-- 📁 muzudho
        +-- 📁 HelloConsoleAppCSharp	    # （後述）フォルダーを新規作成する。
            +-- 📁 HelloConsoleAppCSharp	# （後述）フォルダーを新規作成する。同じ名前が２個入れ子になる。
```


## ［HelloConsoleAppCSharp］プロジェクトのソリューション・エクスプローラーの構成

Visual Studio 2026 のソリューション・エクスプローラーを開いてください。以下のように表示されます。  

```plaintext
📁 ソリューション 'HelloConsoleAppCSharp'
+-- 📁 HelloConsoleAppCSharp				- これはプロジェクトです。
	+-- 📁 依存関係
	+-- 📄 Program.cs						- プログラムのエントリー・ポイント
```


## ［HelloConsoleAppCSharp.Core］プロジェクトの作成

まだ説明していませんが［ライブラリー作成］ができると、あとで便利です。先に準備しておきましょう。  

`📁 ソリューション 'HelloConsoleAppCSharp'` を右クリックして、［追加］＞［新しいプロジェクト］を選択してください。  
フィルターから `C#` を選択して、プロジェクトの種類から `クラス ライブラリ` を選択してください。  
プロジェクト名は `HelloConsoleAppCSharp.Core` としてください。 `.` が含まれて変な感じですが、これでいいです。  

👇 以下のようなフォルダーが作られたはずです。  

```plaintext
📁 ソリューション 'HelloConsoleAppCSharp'
+-- ～中略～
+-- 📁 HelloConsoleAppCSharp.Core
	+-- 📁 依存関係
	+-- 📄 Class1.cs		# 削除してください
```

👆 `HelloConsoleAppCSharp.Core` プロジェクトに最初から入っている `Class1.cs` は（使わないので）削除してください。  

この `HelloConsoleAppCSharp.Core` プロジェクトは、［ライブラリー］を作るのに使います。  


## ［HelloConsoleAppCSharp.Core］プロジェクトを［HelloConsoleAppCSharp］プロジェクトから見れるように設定

`HelloConsoleAppCSharp` プロジェクトの `📁 依存関係` を右クリックして、［プロジェクト参照の追加］を選択してください。  
`HelloConsoleAppCSharp.Core` プロジェクトにチェックを入れて、［OK］をクリックしてください。  

これで、`HelloConsoleAppCSharp` プロジェクトから `HelloConsoleAppCSharp.Core` プロジェクトのコードを参照できるようになりました。  


## 将来的には

将来的には、`HelloConsoleAppCSharp.Core` プロジェクトを NuGet パッケージ化して GitHub Packages で公開するとか、 `*.dll` ファイルとして出力するなどして、他の C# プロジェクトでも使えるようにする予定です。  

以上で、ソリューション一式の作成は完了です。
