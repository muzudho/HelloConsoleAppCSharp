# サービス追加の練習

この記事では、サービスを追加する練習をします。  


## フォルダー構成

```plaintext
📁 HelloConsoleAppCSharp
+-- 📄 Program.cs                    # （後述）既存のファイルを編集
+-- 📄 ProgramService.cs             # （後述）ファイルを新規作成
```

以下のファイルを作成してください。  

📄 `HelloConsoleAppCSharp/ProgramService.cs`:
