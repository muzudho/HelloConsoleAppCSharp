# ライブラリー作成のための準備

まだ説明していませんが［ライブラリー作成］ができると、あとで便利です。先に準備しておきましょう。  


## フォルダーとファイルの構成

```plaintext
📁 ソリューション 'HelloConsoleAppCSharp'
+-- 📁 HelloConsoleAppCSharp		# 既存のプロジェクト
+-- 📁 HelloConsoleAppCSharp.Core	# （後述）プロジェクトを新規作成
```


## プロジェクトの新規作成

`📁 ソリューション 'HelloConsoleAppCSharp'` を右クリックして、［追加］＞［新しいプロジェクト］を選択してください。  
フィルターから `C#` を選択して、プロジェクトの種類から `クラス ライブラリ` を選択してください。  
プロジェクト名は `HelloConsoleAppCSharp.Core` としてください。 `.` が含まれて変な感じですが、これでいいです。  

👇 以下のようなフォルダーが作られたはずです。  

```plaintext
📁 ソリューション 'HelloConsoleAppCSharp'
+-- 📁 HelloConsoleAppCSharp.Core
	+-- 📁 依存関係
	+-- 📄 Class1.cs
```


## 依存関係の設定

`HelloConsoleAppCSharp` プロジェクトの `📁 依存関係` を右クリックして、［プロジェクト参照の追加］を選択してください。  
`HelloConsoleAppCSharp.Core` プロジェクトにチェックを入れて、［OK］をクリックしてください。  

これで、`HelloConsoleAppCSharp` プロジェクトから `HelloConsoleAppCSharp.Core` プロジェクトのコードを参照できるようになりました。  


## 将来的には

将来的には、`HelloConsoleAppCSharp.Core` プロジェクトを NuGet パッケージ化して GitHub Packages で公開するとか、 `*.dll` ファイルとして出力するなどして、他の C# プロジェクトでも使えるようにする予定です。
