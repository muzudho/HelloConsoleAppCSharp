# 矩形塗りつぶしのウォーミングアップ

コンソール画面への Print を、ここではビュー（View）と呼ぶことにします。  
プログラミングでは View と呼ぶと、色んなものを指すので、混乱しないようにしてください。だいたい、目の前に見えているもの、ぐらいの意味です。  

コンソール画面上に、矩形を描画してみましょう。

## 矩形描画メソッドの作成

📄 `HelloConsoleAppCSharp.Core/Infrastructure/MuzConsole.cs`:  

```csharp
namespace HelloConsoleAppCSharp.Core.Infrastructure;

/// <summary>
/// コンソールの操作でよく使う機能をまとめたクラス
/// </summary>
public static class MuzConsole
{


    // ========================================
    // 構成
    // ========================================


    ～中略～


    #region ［矩形描画］


    // 矩形領域の塗りつぶしを行います。


    /// <summary>
    /// 矩形領域を塗りつぶして表示します。
    /// 出力がリダイレクトされていない場合、必要ならコンソール・バッファーを自動的に広げます。
    /// </summary>
    /// <param name="left"></param>
    /// <param name="top"></param>
    /// <param name="width"></param>
    /// <param name="height"></param>
    /// <returns></returns>
    public static async Task FillRectAsync(
         int left,
         int top,
         int width,
         int height,
         ConsoleColor bgColor)
    {
        EnsureBufferSize(left + width, top + height + 1);

        // 処理の後、カーソルの位置を戻す
        await MuzConsole.PreserveCursorPositionAsync(async () =>
        {
            // 指定サイズの矩形領域を塗りつぶします。
            await MuzConsole.RunWithColorAsync(
                bgColor: bgColor,
                onColorChanged: async () =>
                {
                    for (int dy = 0; dy < height; dy++)
                    {
                        Console.SetCursorPosition(left, top + dy);

                        for (int dx = 0; dx < width; dx++)
                        {
                            Console.Write(' '); // ホワイトスペース
                        }
                        Console.WriteLine();    // 改行
                    }
                });


        });
    }

    /// <summary>
    ///     <pre>
    /// 矩形領域を塗りつぶして表示します。
    /// 出力がリダイレクトされていない場合、必要ならコンソール・バッファーを自動的に広げます。
    /// 
    ///     - 色指定も一緒にするぜ（＾～＾）！
    ///     </pre>
    /// </summary>
    /// <param name="left"></param>
    /// <param name="top"></param>
    /// <param name="width"></param>
    /// <param name="height"></param>
    /// <returns></returns>
    public static async Task FillRectAsync(
         int left,
         int top,
         int width,
         int height,
         ConsoleColor? fgColor = null,
         ConsoleColor? bgColor = null)
    {
        await MuzConsole.RunWithColorAsync(
            fgColor: fgColor,
            bgColor: bgColor,
            onColorChanged: async () =>
            {
                await FillRectAsync(left, top, width, height);
            });
    }


    #endregion


}
```

## 仕様メモ

- `width <= 0` のときは何も描画しません。  
- `height <= 0` のときも何も描画しません。  
- 負の値を使って反対方向へ塗りつぶす機能は、ここではサポートしません。  
- 出力がリダイレクトされていない場合、必要ならコンソール・バッファーを自動的に広げます。
