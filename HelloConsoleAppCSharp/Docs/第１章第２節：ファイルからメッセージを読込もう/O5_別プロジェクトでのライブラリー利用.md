# 別プロジェクトでのライブラリー利用

今回、HelloConsoleAppCSharp.Core プロジェクトで作成したクラスは、別のプロジェクトでも利用できます。  

主に２通りあり、その方法の１つを紹介します。  

* `*.dll` ファイルをコピーして利用する方法。(こっちを説明します）
* NuGet パッケージを作成して利用する方法。


## フォルダーとファイルの構成

```plaintext
📁 ソリューション 'HelloConsoleAppCSharp'
+-- 📁 HelloConsoleAppCSharp
    +-- 📁 bin
        +-- 📁 Debug
            +-- 📁 net10.0
                +-- 📄 HelloConsoleAppCSharp.Core.dll    # これがライブラリーのファイルです。
```

👆　この `*.dll` ファイルを、別ソリューションのフォルダーへ持っていく（後述）わけです。  


## 別ソリューションのフォルダーとファイル構成

```plaintext
📁 ソリューション 'HelloConsoleAppCSharp'
+-- 📁 HelloConsoleAppCSharp
    +-- 📁 Lib                                   # フォルダーを新規作成
        +-- 📄 HelloConsoleAppCSharp.Core.dll    # ここに置く
```

👆　手動でコピー貼り付けしてください。  

あとは、プロジェクトからこの `HelloConsoleAppCSharp.Core.dll` を参照するように設定すれば、別プロジェクトから HelloConsoleAppCSharp.Core のクラスを利用できるようになります。
