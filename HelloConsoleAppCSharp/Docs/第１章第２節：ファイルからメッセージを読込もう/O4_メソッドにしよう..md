# 第１章第２節：ファイルからメッセージを読込もう　＞　メソッドにしよう

［メッセージ・ファイルの読込］（これを［ロード］と呼ぶとします）と、［キーを指定してメッセージを取得］（これを［ゲット］と呼ぶとします）の２つの手順に分かれていると、
［ゲット］をする前に、最初に［ロード］をしておく、というのが面倒くさくなってしまいます。  

どうしても手順は１つにしたいです。  

そこで、［ゲット］の１手順の中に、  
［ロード］をする必要があるときは、［ロード］も一緒にするようにし、必要がなければしない、というようにしてみましょう。  

すると、［ロード］をする必要はないが［ロード］してほしい、というときどうにかする方法が欲しくなります。  
そんなときは［ゲット］に［forceLoad = true］（強制的にロードする）という引数を追加することで対応することにします。

## 既存のサービスの編集

📄 `HelloConsoleAppCSharp.Core/Features/Messages/MuzMessagesService.cs`:  

```csharp


～前略～


public class MuzMessagesService
{


    // ========================================
    // 窓口データメンバー
    // ========================================


    ～中略～


    /// <summary>
    /// 見つからないときのデフォルトメッセージ
    /// </summary>
    public string DefaultMessage { get; set; } = "メッセージが見つからないぜ（＾～＾）！";


    // ========================================
    // 窓口メソッド
    // ========================================


    /// <summary>
    /// メッセージの取得
    /// </summary>
    /// <param name="key">メッセージのキー</param>
    /// <param name="forceLoad">キャッシュを捨てて再読込したいとき真</param>
    /// <returns>メッセージ</returns>
    public string GetMessage(
        string key,
        bool forceLoad = false)
    {
        // 強制読込（空のときも読込）
        if (!this.MessageCache.Any() || forceLoad)
        {
            // 設定されているファイルパスを、実行ファイルのディレクトリを基準にした絶対パスに変換します。
            var baseDirectory = AppDomain.CurrentDomain.BaseDirectory;
            var absolutePath = Path.Combine(baseDirectory, this.FilePath);
            this.MessageCache = MuzMessagesHelper.GetMessagesAsDictionary(absolutePath);
        }

        return this.MessageCache.GetValueOrDefault(key, this.DefaultMessage);
    }


}
```
