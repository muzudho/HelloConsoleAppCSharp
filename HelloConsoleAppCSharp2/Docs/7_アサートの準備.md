# アサートの準備

変数の中身が予想通りであるかを確認する［アサート］というテクニックを使えるよう、準備します。  


## using ディレクティブの追加

C# 言語では最初から［アサート］が用意されています。  

```csharp
using System.Diagnostics;
```

👆 上記をファイルの冒頭に追加してください。  


## アサートの例

```csharp
// デバッグビルドのときだけ有効になるやつ（おすすめ）
Debug.Assert(banana == 10, "バナナは１０本でないといけないぜ");

// リリースビルドのときも有効になるやつ（あまりおすすめしない）
Trace.Assert(banana == 10, "バナナ１０本必須");
```

以上で、［アサートの準備］は完了です。
